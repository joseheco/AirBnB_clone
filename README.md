---

Coding Style
-----------
All python files are interpreted and compiled on Ubuntu 20.04 using python3 and PEP 8 style.


---

Usage
-----------
<p>Interactive Mode</p>

```
$./console.py
(hbnb) <command>
```

<p>Non-Interactive Mode</p>

```
$ echo "<command>" | ./console.py
```

---
Example
-----------

```
$./console.py
(hbnb) all....

```
